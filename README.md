## 筆記
### IntelliJ：關閉自動補全提示
在學習基礎語法時，如果不希望 IntelliJ 一直自動跳提示，可以到:<br>
`File → Settings → Editor → General → Inline Completion`<br>

將以下選項關閉：
- Enable automatic completion on typing<br>
關閉輸入時的自動補全。
（打字不會跳出灰色預測或提示選單，需要補全時才手動用快捷鍵。）

- Enable multi-line suggestions<br>
關閉多行預測補全。
（避免 IntelliJ 一次預測整段程式碼。）
